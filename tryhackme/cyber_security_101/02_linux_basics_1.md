
## Linux Basics 1

### Commands:


**echo** : Output any text that we provide

**whoami** : 	Find out what user we're currently logged in as

**ls** : List contents of current directory

**ls <directory_name>** : List contents of directory without navigating to it

**cd** : Change directory

**cat** :	Concatenate

**cat <directory_name/file_name>** : Concatenate without navigating to directory

**pwd** :	Print working directory

**find** : Search through current directory

**find -name <file_name>** : Search for specific file in current directory

**find -name "*.txt"** : Search for all .txt files in current directory

**grep** : Search through contents of files for specific value

**grep -R <directory_name>** : Search recursively through all files and subdirectories

---

## Operators


**&** : Allows you to run commands in the background of your terminal

**&&** : Allows you to combine multiple commands together in one line of your terminal

**>** : This operator is a redirector - meaning that we can take the output from a command (such as using cat to output a file) and direct it elsewhere

**>>** : This operator does the same function of the > operator but appends the output rather than replacing (meaning nothing is overwritten)
