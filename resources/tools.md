## Tools

**Nikto:**
Nikto is an open-source, command-line web server scanner used by security professionals to identify vulnerabilities, dangerous files, outdated server software and misconfigurations.

**Gobuster:**
Gobuster is a software tool for brute forcing directories on web servers.

**Wfuzz:**
Wfuzz is an open-source web application fuzzer designed for brute-forcing web applications.

**DIRB:**
DIRB is a powerful, open-source web content scanner designed to uncover hidden files, directories, and resources within a web server.
